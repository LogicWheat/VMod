# Snap Mode

In certain modes (Connection/Hydraulics) there exists a "Snap Mode", usually as a secondary toolgun mode. This mode allows you to snap one ship to another. It also respects "HitPos Modes".

To use:
1. Set position of the first ship.
2. Set to which position snap second ship.
3. Use scroll wheel to rotate first ship, before finalizing.

Custom Parameters:
* **Distance From Block** - Final distance from block.
* **Snap Mode Scroll Step** - Scroll wheel step in degrees.
